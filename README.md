Alephyx Exchanche Beta - elCDPI

To use this basic HTML exchange, complete the following steps in order.
1. Install Metamask in Google Chrome.
2. Set Metamask's net to Sepolia.
3. Click "Add token" and type the contract direction "0xFF2a2F72aC743db64E645694479548f821363e97". Then, confirm it says "Alephyx". You can search the source code and other data in:
https://sepolia.etherscan.io/token/0xff2a2f72ac743db64e645694479548f821363e97
4. Download "index.html". Run the command "python3 -m http.server 8000" in the same directory where you downloaded "index.html".
5. Go to Google Chrome and type in the upper search bar "http://localhost:8000/index.html".
6. Connect the exchange to your Metamask wallet. Insert an amount of sepolia ETH (do not set Metamask's net to Ethereum mainnet).
7. Click buy, if you completed every step, it should work fine.
